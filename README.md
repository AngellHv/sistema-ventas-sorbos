SISTEMA DE VENTAS SORBOS
Sorbos es un sistema de escritorio desarrollado en Java que permite gestionar la venta de licores, facilitando el registro de productos, control de stock y registro de ventas de manera sencilla.
Este proyecto ha sido desarrollado como parte del curso de Algoritmos y Estructuras de Datos, con el objetivo de aplicar conceptos de programación, bases de datos y diseño de sistemas.

1. Objetivo
Desarrollar un sistema que permita administrar productos y ventas, asegurando un control básico de inventario y registro de transacciones.

2. Funcionalidades
•	Inicio de sesión de usuario
•	Registro, edición y eliminación de productos
•	Registro de ventas
•	Visualización del historial de ventas
•	Actualización automática del stock

3. Tecnologías utilizadas 
•	Java
•	NetBeans
•	PostgreSQL
•	GitHub
