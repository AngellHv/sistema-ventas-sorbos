# Flujo del Sistema - Sorbos

##   Proceso general

1. El usuario inicia sesión en el sistema.
2. Accede al menú principal.
3. Registra o consulta productos.
4. Realiza una venta seleccionando productos.
5. El sistema calcula el total de la venta.
6. Se registra la venta en la base de datos.
7. El sistema descuenta automáticamente el stock.
8. El usuario puede visualizar el historial de ventas.

---

##   Consideraciones

 No se permite vender productos sin stock.
 Todas las ventas quedan registradas.
 El sistema mantiene consistencia en los datos.
