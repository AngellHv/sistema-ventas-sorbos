# Requerimientos del Sistema - Sorbos

##   Descripción general

El sistema "Sorbos" es una aplicación de escritorio orientada a la gestión de ventas de licores. Permite administrar productos, registrar ventas y mantener un control básico de inventario.

---

##   Objetivo

Facilitar el registro y control de ventas, asegurando la correcta gestión del stock de productos.

---

##   Requerimientos Funcionales

* El sistema debe permitir el inicio de sesión de usuarios.
* El sistema debe registrar nuevos productos.
* El sistema debe editar y eliminar productos existentes.
* El sistema debe listar los productos disponibles.
* El sistema debe registrar ventas.
* El sistema debe actualizar el stock automáticamente al realizar una venta.
* El sistema debe mostrar el historial de ventas.

---

##   Requerimientos No Funcionales

* Interfaz gráfica sencilla e intuitiva.
* Tiempo de respuesta rápido en operaciones básicas.
* Uso en entorno de escritorio (Java NetBeans).
* Conexión a base de datos PostgreSQL.

---

##   Alcance

El sistema no contempla:

* Facturación electrónica
* Múltiples roles de usuario
* Integración con sistemas externos
