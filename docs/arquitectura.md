# Arquitectura del Sistema - Sorbos

##   Enfoque

El sistema está organizado en una estructura por capas para mejorar la claridad y mantenimiento del código.

---

##                       Capas del sistema

###  Modelo (Model)

Representa las entidades del sistema

 Producto
 Usuario
 Venta

---

###   Acceso a Datos (DAO)

Encargado de la comunicación con la base de datos

 Consultas SQL
 Inserciones, actualizaciones y eliminaciones

---

###   Vista (GUI)

Interfaz gráfica desarrollada en Java NetBeans

 Formularios (JFrame)
 Tablas (JTable)
 Botones y campos de texto

---

##   Ventajas

 Separación de responsabilidades
 Código más organizado
 Mayor facilidad de mantenimiento
